To run this project, you will have to downlaod Unreal Engine V4.11.
You will also need Visual Studio 2015 and the "Common tools for C++" (or something like that) from the VS 2015 install.


Right click the .uproject file and click "Generate visual studio files" and once that has completed just open the .uproject file to open the editor, or right click -> launch game to just run the game.
