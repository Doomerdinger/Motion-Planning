Javascript A* Demonstration

Try the A* visualizer here!
Usage: left click to create obstacles (dark grey), right click to create objectives (green). Clicking when there is not a start node (red) will create one.
Upon clicking "run", you will see a visualization of the A* algorithm attempting to find the optimal path through all objective points. The slider bar at the bottom of the page controls how quickly the visualization runs.
